# **GIT**

Push de archivo locales hacia repo publico en github:

* Inicializar:

  ```
  git init
  ```
* Agregar repo local:

  ```
  git add .
  ```
* Lanzar commit inicial:

  ```
  git commit -m "primer readme"
  ```
* Agregar repo remoto:

  ```
  git remote add origin https://github.com/abelucci/GIT.git
  ```
